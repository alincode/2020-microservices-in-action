# 微服務之間的通訊

單體式系統架構差成多個為服務後，第一個要面臨的問題就是，這些服務之間怎麼溝通。

我們有幾項需要考量：

- 通訊協定
  - 同步通訊協定
    - RESTful
    - RPC
  - 非同步通訊協定
    - Message Queue (MQ)
      - pubsub
      - Kafka
      - RabbitMQ
- 訊息保證送達
- 訊息傳送的效率
- 版本管理
  - 破壞性變更
